# Notes App

A simple Note Taking app built with React.

## Authors

- [@kamalnayan10](https://github.com/kamalnayan10)

## Running Locally

Running the project locally with `npm`

```bash
  npm i
  npm run start
```
    
## Features

- Local storage support
- Minimalistic
- PWA support
